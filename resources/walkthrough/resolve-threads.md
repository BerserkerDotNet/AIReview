## Resolving Threads with Copilot

Once the plugin is installed, Copilot can resolve your open review threads automatically.

### How it works

1. Copilot reads open threads from `.vscode/.ai-review.json`
2. For each thread, it analyzes the code context
3. It implements the fix, adds a reply comment, and marks the thread as resolved

### Try it

Ask Copilot: *"Resolve my open review comments"*
