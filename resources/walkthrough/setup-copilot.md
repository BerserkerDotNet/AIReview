## Setting Up the Copilot Plugin

The **resolve-comments** skill lets Copilot automatically resolve your review threads.

### Install via Copilot CLI

Run these commands in the Copilot CLI:

```
/plugin marketplace add BerserkerDotNet/AIReview
/plugin install feedback-resolver@ai-changes-review-marketplace
```

### Or use the command palette

Run **AI Changes Review: Setup Copilot Plugin** from the command palette (`Ctrl+Shift+P`) for guided installation.
